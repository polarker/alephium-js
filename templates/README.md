# My dApp

## Install

```
npm install
```

## Compile

Compile the TypeScript files into JavaScript:

```
npm run compile
```

## Start a devnet

```
npm run devnet:start // this will start a devnet for smart contract tests
```

## Stop/restart devnet

```
npm run devnet:stop
npm run devnet:restart
```

## Testing

```
npm test
```
